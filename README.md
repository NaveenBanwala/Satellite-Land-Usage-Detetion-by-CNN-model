# 🌍 Satellite Land Usage Detection using CNN

## 📌 Project Overview

This project uses a **Convolutional Neural Network (CNN)** to analyze satellite images and detect land usage changes over time. It compares “before” and “after” imagery to identify modifications such as urban expansion, vegetation loss, or land transformation.

---

## 🚀 Features

* 🛰️ Satellite image analysis
* 🔍 Land use change detection (before vs after)
* 🤖 Deep learning model using CNN
* 📊 Visual comparison of land transformations

---
